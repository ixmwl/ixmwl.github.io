自从计划着要搭建自己的博客,就满网海里面搜索博客搭建之法,翻阅了无数前人写下的“武功秘籍”,虽然天资愚钝,但皇天不负有心人,终于按部就班的练成了神功,并且自己还有所感悟,所以就此记录下,希望也能福泽世人。经过多方探索,最终搜集了四部绝世秘籍,都可助大家筑基入门:

  • Jekyll (https://jekyllrb.com/)
  • hugo (http://www.gohugo.org/)
  • django (https://www.djangoproject.com/)
  • hexo (https://hexo.io/)

最后苦苦纠结选择了最适合我的hexo(主要是因为他博客主题多),那么话不多说,下面就开始抛砖引玉。
