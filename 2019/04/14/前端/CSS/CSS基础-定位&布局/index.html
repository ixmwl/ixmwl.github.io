CSS基础-布局&定位

发布 : 2019-04-14 分类 : 前端 浏览 :

布局是什么？

CSS布局就是将不同的元素按照一定的规则放置在页面上、窗口上的不同位置，因此经常会用到一些内容、填充、边框、边界等属性。CSS页面布局技术允许我们拾取网页中的元素，并且控制它们相对正常布局流、周边元素、父容器或者主视口/窗口的位置。关于页面窗口，我们可以把它当作一个盒子，我们的布局就可以理解为我们需要把不同的区块放置在这个盒子的不同位置，此处简称盒子模型。

布局常见属性

关于布局中我们经常见到的两个属性就是：margin和padding，两者都是边距的含义，关键问题得明白是什么相对什么的边距，padding是控件的内容相对控件的边缘的边距，margin是控件边缘相对父空间的边距．具体的区别盗用别人的图解释下：


margin

margin在一个声明中设置所有外边距的属性，属性可以有[1～4]个不等,不同个数所代表的意义也不一样。

四个值：margin:10px 5px 7px 20px 顺序是（上、左、下、右）
三个值：margin:10px 15px 20px 顺序是（上、左右、下）
两个值：margin:10px 20px 顺序是（上下、左右）
一个值：margin:10px 代表四个值都是10px

padding

padding在一个声明中可以设置所有内边距的属性，格式与margin顺序一致
