JavaScript基础--数据类型&类型转换

JavaScript（简称“JS”） 是一种具有函数优先的轻量级，解释型或即时编译型的编程语言。虽然它是作为开发Web页面的脚本语言而出名的，但是它也被用到了很多非浏览器环境中，JavaScript 基于原型编程、多范式的动态脚本语言，并且支持面向对象、命令式和声明式（如函数式编程）风格。

数据类型

Number-数字类型

在其他编程语言中，经常会有整型、浮点型的说法，也就是区分是否带小数点的数字，而在js中没有严格区分，也就意味着15 === 15.0

console.log(15 === 15.0);//打印true

而且需要注意的是，浮点型在参加运算的时候，可能并不是很精确，比如

console.log(1-0.7); // 打印0.3000000000000000004
console.log(1-0.9); // 打印0.0999999999999999998

String-字符串类型

使用双引号或者单引号括起来的数据类型，例如：

var userName = "Tim";

字符串类型特点：

  1. 单引号不可直接嵌套单引号、同理双引号也一样，但是单引号和双引号是可以相互嵌套的
  2. 字符串有长度属性（这个oc、java里也有长度属性）
  3. 字符串包含0个或者多个字符，如果引号内不写则代表空字符串

Boolean-布尔类型

同oc、java等语言，js也存在布尔类型，代指真假、是否，关键字是true和false

null

一个表明null值的特殊关键字。之前说过js是区分大小写的，所以null,Null和NULL或者其他变量意义是不一样的。null通常表示此处没有对象即该处不应该有值。

undefined

表示变量未定义时的属性，可以理解为未定义或者缺少值，即此处应该有值但没给
上面说到打印的时候提过，未定义的变量打印会是undefined

类型转换

强制转换

强制类型转换就是使用固定的方法强制把一种类型的转换成另一种类型

强制转换成数字类型

纯字符串数字

var a = "14";
a = parseInt(a); 
console.log(typeof a);//Number

带小数点的纯字符串数字

var a = "13.23";
a = parseInt(a); 
console.log(a);//输出13

字符串和数字的混合

var a = "13asdasds45";
a = parseInt(a); 
console.log(a);//输出13

不以数字开头的混合体

var a = "qw341233";
a = parseInt(a); 
console.log(a);//输出NaN 表示不是一个数字

总结：parseInt函数将变量转换成一个整数，具体分以下几种情况：

  1. 如果变量值为字符串数字，直接转成整数
  2. 如果变量值既有数字又有字符串，以数字字符串开头，那么转换时会把前面的字符串数字转成整数并舍去后面的字符串
  3. 如果变量值既有数字又有字符串，不以数字字符串开头，那么转换成NaN，表示转换结果不为数字
  4. 如果是带小数点的纯字符串数字，只会保留小数前的转成整数，其实此处可以理解为第2点

强制转换成字符类型

强制转换成字符类型，就比较简单，记住String关键字即可

var a = 12;
a = String(a);
console.log(typeof a);//String

强制转换成布尔类型

记住真假值即可，因为大多数情况下都是true，只有少数为false

var a = 12;
var b = 0;
a = Boolean(a);//true
b = Boolean(b);//false

自动转换

在js运行过程中，会根据上下文语义，自动将类型转换成统一类型进行运算，例如：

var a = "5";
var b = 4;
var c = a-b;
console.log(c);//输出1

可见此处根据减法语义，将a转换成number类型与b进行减法操作。
